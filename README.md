# Commodity Quant Universal Framework — DEPRECATED

**This repo has been split into two framework-specific versions. Please use the appropriate one:**

| Version | Paradigm | Repo |
|---------|----------|------|
| **OpenAI Agents** | Agent handoff + function tools + Runner | [seraon/commodity-quant-openai](https://github.com/seraon/commodity-quant-openai) |
| **Claude Code** | System prompt + tool use loop + file state | [seraon/commodity-quant-claude](https://github.com/seraon/commodity-quant-claude) |
| **WorkBuddy** | WorkBuddy expert plugin | [seraon/commodity-quant](https://github.com/seraon/commodity-quant) |

All three repos share the same v3.8 pipeline, chain security, evidence quality scoring, and data iron law v2.2.1.

The difference is how agents are defined and orchestrated — choose based on your framework.
